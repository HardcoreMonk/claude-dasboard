01 Real-time Monitoring

JSONL file changes are detected within seconds and pushed to all connected browsers.

WebSocket Live Update

watchdog (inotify) + 30s safety poll detects JSONL changes. Batched via 800ms debounce, pushed to all WS clients.

watcher.py + app.js

4-Phase Project Status

Waiting for Input (amber) / Permission Prompt (amber) / Tool Running (cyan) / Agent Working (blue). Auto-detected from message type + 15s timeout.

state.idleProjects

Audio Chime Notification

Web Audio API plays C5-G5 two-note chime when a project enters idle state. Configurable on/off per user.

Web Audio API

Connection Status

WS connected/disconnected indicator. Auto-reconnect with exponential backoff. 15s persistent banner on sustained disconnect.

ConnectionManager

13 UI / UX

Dark / Light Theme

Toggle with WCAG AA 4.5:1 contrast on all text. body.theme-light class.

Responsive

3-tier breakpoints: 640px, 480px, 360px. Mobile-friendly tables.

Keyboard Shortcuts

g+o/b/s/c view switch, Cmd+K command palette, vim-style j/k.

Command Palette

Fuzzy search across views, projects, settings. 30s TTL cache.

Toast Notifications

Success/warning/error with undo support. 3-strike persistent banner.

Animations

fadeInUp + blur(3px) + stagger. Spring transitions. Respects prefers-reduced-motion.

Accessibility

Focus traps on modals, tabindex + role on interactive elements, aria-live regions.

Design System

Supanova: #0a0a0a bg, Emerald #34d399 accent, double-bezel cards, Pretendard font, Iconify Solar icons.

NEW Public Landing Pages /landing/

Self-contained intro pages for unauthenticated visitors. Bundled with Supanova design patterns (Pretendard, Emerald accent, Double-Bezel cards). Served from landing-pages/ with full auth bypass and path-traversal guard.

Combined

A+B+C integrated story: 01 Local → 02 Team → 03 Proof. Features editorial hero, dashboard mockup, architecture diagram, 눈대중 vs 실측 comparison table, dual quotes, massive CTA.

Variant A — Editorial

Minimal editorial tone, massive typography, extreme whitespace. Zig-zag features. For senior-developer taste.

Variant B — Data-Viz

Split hero with live dashboard mockup, metric bar, weekly heatmap, model/stop-reason donuts, comparison table. For power users.

Variant C — Multi-node

Team/organization angle: live-nodes panel, collector architecture diagram, audit log console, team budget, security trust cards.

Standalone HTML

No build step. CDN only (Tailwind, Pretendard, Geist, Iconify Solar, Motion One). DOM-API rendering (XSS-safe).

Dashboard Integration

Login page footer links to /landing/. Auth middleware bypasses /landing/*. Path traversal guard matches /static/* pattern.
